## ⚡ Quick Start

### Prerequisites

- **Node.js** ≥ 18
- **npm** ≥ 9

### Installation

```bash
# Clone the repository
git clone https://github.com/SurajMoreTech/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000/portfolio](http://localhost:3000/portfolio) in your browser.

### Production Build

```bash
# Build static export
npm run build

# Output is in the ./out directory
```

## 🌍 Deployment

This project auto-deploys to **GitHub Pages** via GitHub Actions on every push to `main`.

The workflow (`.github/workflows/deploy.yml`) handles:
1. Installing dependencies
2. Building the Next.js static export
3. Deploying the `./out` directory to GitHub Pages

> **No manual deployment needed** — just push to `main` and it's live!
